# Feature Request: Landing 页面相关链接页整体补完

**Created**: 2026-03-12
**Source**: User Request

---

## 需求描述
补全 landing 页面中所有已暴露或已规划的相关链接页面，避免出现入口可见但目标页面缺失、占位或不可访问的情况。

## 当前行为
- landing 页面存在若干相关链接入口
- 部分链接对应页面缺失，或仅有占位内容
- 用户点击后无法完成信息获取或后续转化路径

## 期望行为
- 对 landing 页面中的相关链接进行完整梳理
- 为缺失链接补齐对应页面
- 页面内容、导航关系和跳转路径保持完整可用
- 对无短期落地价值的链接，应下线入口或提供明确状态说明，避免死链

## 相关组件
- Frontend: `auth9-portal` landing 页面及其相关路由
- Content / UX: 页面信息架构、导航与转化路径

## Severity
Medium
